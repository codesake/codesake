codesake

codesake is a security source code scanning engine. It's used as core engine in codesake.com application security portal with a closed knowledge base inside the web application itself.

codesake gem can be used in a security source code review to scan sources with regular expressions in order to detect insecure software patterns.

Installation

Add this line to your application's Gemfile:

gem 'codesake'

And then execute:

$ bundle

Or install it yourself as:

$ gem install codesake
